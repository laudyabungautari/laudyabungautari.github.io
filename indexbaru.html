Tutorial Hosting Website di GitHub

  1. Buka https://github.com lalu daftar akun GitHub.
  2. Buat repository baru dengan nama: username.github.io
  3. Upload file website-mu (seperti index.html, style.css, dll).
  4. Aktifkan GitHub Pages melalui tab Settings → Pages.
  5. Pilih branch: main dan folder: /root.
  6. Setelah beberapa detik, buka: https://username.github.io
